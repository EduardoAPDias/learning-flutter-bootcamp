Flutter Bootcamp: Dart e Flutter

Este repositório documenta o progresso e os projetos práticos desenvolvidos durante o curso "The Complete Flutter Development Bootcamp with Dart". O foco é o aprendizado da linguagem Dart e do framework Flutter, abrangendo desde a construção de interfaces até a lógica de aplicação.
Projetos Desenvolvidos
Projeto	Descrição	Conceitos Aplicados
Mi Card	Cartão de visitas pessoal	Stateless Widgets, Layouts (Row/Column), Assets.
Dicee	Simulador de dados aleatórios	Stateful Widgets, Lógica Dart, Gestos.
Magic 8-Ball	Aplicativo de tomada de decisão	Manipulação de estados e interatividade.
Xylophone	Instrumento musical interativo	Integração de pacotes externos (audioplayers).
